# Hierarchical Mergers of Binary Black Holes

This project studies hierarchical mergers of binary black holes (BBHs) in dense stellar environments using simulated data.

The objective is to analyse how different environments (nuclear star clusters, globular clusters, and young star clusters) affect the formation and evolution of multi-generation black holes.

Methods include statistical analysis of simulated BBH populations and machine learning techniques to identify the most relevant features driving merger outcomes.

The project provides a data-driven characterisation of hierarchical merger processes and the key factors governing the evolution of complex dynamical systems.
